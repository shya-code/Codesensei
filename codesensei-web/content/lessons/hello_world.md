Python is a very simple language, and has a very straightforward syntax. It encourages programmers to program without boilerplate (prepared) code. The simplest directive in Python is the `print` function — it simply prints out a line (and also includes a newline).

This course uses **Python 3**. In Python 3, `print` is a function and must be invoked with parentheses.

To print a string in Python 3, just write:

```python
print("This line will be printed.")
```

### Indentation

Python uses indentation for blocks, instead of curly braces. The standard Python indentation is **four spaces**. For example:

```python
x = 1
if x == 1:
    # indented four spaces
    print("x is 1.")
```
